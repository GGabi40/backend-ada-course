# BACKEND con ADATECH

## Arrancar con npm:
    npm init
o utilizar:

    npm init -y

después utilizar:

    npm install

----

## Paquetes utilizados:

* dotenv: Utilizado para manipular variables de ambiente

        npm install dotenv

        npm docs dotenv

    - Crear un archivo __.env__ y agregarlo al __.gitignore__ para evitar que se haga pública la información del archivo.
